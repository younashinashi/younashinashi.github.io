Vaulta 利用規約

最終更新日：

第1条（適用）

本規約は、younashinashi（以下「開発者」といいます）が提供するスマートフォンアプリ「Vaulta」（以下「本アプリ」といいます）の利用条件を定めるものです。本アプリを利用する者（以下「ユーザー」といいます）は、本規約に同意のうえ本アプリを利用するものとします。

第2条（本アプリの概要）

1. 本アプリは、訪問介護等の現場業務におけるスケジュール管理および現場メモ・モニタリングをサポートすることを目的とし、主として次の機能を提供します。

  • （1）訪問予定メールの貼り付け解析による訪問時間・移動時間・空き時間等の一覧表示機能
  • （2）Google Maps Platform（API）を用いた移動時間の取得および出発時刻・帰宅時刻等の目安表示機能
  • （3）利用者ごとの当日メモ・体調等の現場メモ記録・閲覧機能
  • （4）利用者情報のローカル管理機能（必要に応じた追加・編集等）
  • （5）ライトモード／ダークモード等の表示切替機能
  • （6）アプリ起動時ロック、4桁 PIN コード認証および端末の生体認証と連携したロック機能
  • （7）暗号化されたバックアップ・復元等のセキュリティ機能

2. 本アプリはユーザー登録を要せず、無料で利用できます。

3. 本アプリは、クラウド同期や外部サーバー保存機能を提供せず、データは端末内で完結する設計としています。また、ファイルとしてのエクスポート等による外部出力機能は提供しません。

第3条（利用対象者）

1. 年齢・国籍・居住地等の制限は設けませんが、本規約および適用される法令を遵守できる方のみ、本アプリを利用できます。

2. 業務で本アプリを利用する場合、ユーザーは、自身が所属する事業所・会社等の就業規則、情報セキュリティポリシーその他社内ルールとの適合性について、自らの責任で確認したうえで利用するものとします。

第4条（料金）

本アプリは無料で提供され、アプリ内課金および利用料金は発生しません。ただし、ユーザーが本アプリの利用に伴い別途契約した通信回線・端末・外部サービス（Google Maps Platform 等）にかかる費用は、すべてユーザーの負担とします。

第5条（ユーザー情報の取扱いおよびセキュリティ）

1. ローカル保存

（1）本アプリは、ユーザーが入力した訪問予定・利用者情報・現場メモ等のデータをインターネット経由で開発者サーバーへ送信・保存せず、原則として端末内にローカル保存します。

（2）ユーザーが本アプリをアンインストールした場合、アプリが管理するデータは端末内から削除されます。ただし、OS の仕様やバックアップ機能等により、端末あるいはクラウドバックアップ上に複製・キャッシュが残存する可能性があり、開発者はそれらについて管理・保証しません。

2. 暗号化およびロック機能

（1）本アプリは、端末のストレージ暗号化を前提としつつ、アプリ内データの一部およびバックアップデータについて、AES-256-GCM 等の方式により暗号化を行います。復号にはユーザーが設定するパスフレーズ等により導出された鍵が必要となります。

（2）鍵の導出には PBKDF2-HMAC-SHA256 などの鍵導出関数とソルトを用いて、総当たり攻撃に対する耐性を高める設計としています。

（3）暗号化処理では毎回ランダムな初期化ベクトル（IV）を使用し、GCM の認証タグにより改ざん検知を行います。

（4）本アプリは、アプリ起動時ロック、4桁 PIN コードおよび端末 OS が提供する生体認証機能と連携したロック機能を提供します。PIN コードや生体情報の管理・第三者への漏えい防止はユーザー自身の責任で行うものとします。

（5）暗号化・ロック機能は安全性向上のための措置であり、あらゆる攻撃や不正アクセスからの完全な防御を保証するものではありません。

3. スクリーンショット・画面録画・タスクサムネイル

（1）本アプリは、個人情報保護の観点から、可能な範囲でスクリーンショット、画面録画、タスクサムネイル（アプリ履歴画面のサムネイル表示）を抑止・マスキングする設計を採用しています。

（2）ただし、端末や OS の仕様・バージョン、サードパーティ製アプリ等の影響により、これらの抑止機能が常に完全に動作することを保証するものではありません。ユーザーは、撮影・録画が行われる可能性が完全には排除できないことを理解したうえで利用するものとします。

4. 通知表示・クリップボード

（1）本アプリは、通知に氏名・住所等のセンシティブな情報が直接表示されないよう配慮し、利用者 ID 等の最小限の情報表示にとどめる設計としています。ただし、OS の通知設定や他アプリとの連携状況により、表示内容が変化する場合があり、開発者はそれを完全には制御できません。

（2）本アプリは、氏名・住所等が OS のクリップボードに長時間残存しないよう配慮した UI 設計を採用していますが、ユーザーの操作方法や端末設定によりクリップボードへコピーされる可能性を完全に排除するものではありません。

5. 電話発信機能

本アプリから電話番号をタップした場合、標準の電話アプリのダイヤラ画面を開くだけであり、自動発信や連絡先アプリへの自動保存は行いません。実際の発信操作および発信内容は、すべてユーザーの責任となります。

6. ログ出力

開発者は、クラッシュレポートやデバッグログ等において、可能な限り氏名・住所等の個人を特定し得る情報を含めないよう配慮します。ただし、OS やライブラリの仕様により、一部情報が自動的に送信・記録される場合があり、開発者はそれらについて完全な非包含を保証するものではありません。

7. 第三者 SDK・外部サービス

（1）本アプリは、広告 SDK や Firebase 等、一般的なトラッキング・広告配信用の第三者 SDK を利用していません。

（2）ただし、地図表示および移動時間取得のために Google Maps Platform（API）を利用します。Google Maps Platform は、Google が定める利用規約およびプライバシーポリシーに基づき、位置情報等を処理する場合があります。ユーザーは、当該規約・ポリシーに同意したうえで本アプリを利用するものとします。

（3）ユーザーが自身で Google Maps Platform の API キーを取得・設定する方式の場合、その契約・課金・利用制限等はすべてユーザーの責任において管理するものとし、開発者はこれに起因する料金・制限・利用停止等について責任を負いません。
